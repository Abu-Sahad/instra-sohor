How Does JavaScript Work

JavaScript is a prewritten language that runs on an Internet browser on a visitors computer.If someone wants to run it then a web developer has to first place a JavaScript code on to HTML pages.Which requests an HTML page along with JavaScript in it,then the script is sent to the users browser and the Internet browser deals with it from then on.If it goes right, the JavaScript code will run automatically without any action.Depending on the user's wish for the javaScript code to run,it may start with a web page open or when a click or drag is made on a page element or when a form is submitted this can be specified from the outset.When it is placed within the HTML page and is supported by a users Internet browser then it will create a seamless and functional experience for your users.

What Does JavaScript Event Loop do

The event loop is mostly used in asynchronous programming.We know js executes all operations on a single thread.There are two main two keywords in the event loop one is stack and another is an event.Stack works for keeping track of all the operations in line to be executed when the function is finished, it is pooped from the stack.event queue works for sending new functions to the stack for processing, which follows the queue data structure to maintain the correct sequence in which all operations should be sent for execution. When an async function is called, it is sent to a browser API.These are APIs built into the browser from the beginning. Depending on the command received from the call stack, the API starts its own single-threaded operation.

Difference between local storage vs session storage

The main difference between local storage and session storage is that the data in the local storage has to be deleted manually but the data of the session store is cleared when the browser or tab is closed.Local Storage does not work for any specific tab, works for any tab, but session Storage works for any specific tab, and data is lost when the tab is closed.Local storage limit is about 5MB,And Session Storage limit is about 5-10MB.Local storage and Session storage data is doesn't transferred to the server.local storage web page can be viewed simplicitily for greater storage capacity,and session storage similar to local storage
